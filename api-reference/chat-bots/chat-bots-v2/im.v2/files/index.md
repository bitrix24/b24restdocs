# Files: Overview of Methods

This section contains methods for uploading files to the chat and obtaining a download link for the file on behalf of the user or application.

> Quick navigation: [all methods](#all-methods)

## How to Work with Files

- Use [im.v2.File.upload](./file-upload.md) to upload a file to the chat.
- Use [im.v2.File.download](./file-download.md) to obtain a download link for an already uploaded file.

## Overview of Methods {#all-methods}

> Scope: [`im`](../../../../scopes/permissions.md)
>
> Who can execute the methods: a user or an application with access to the messenger
