# Task Templates: Overview of Methods

Task templates allow you to describe a typical task and create tasks according to unified rules.

> Quick navigation: [all methods and events](#all-methods) 
> 
> User documentation: [How to work with task templates](https://helpdesk.bitrix24.com/open/27889912/) 

## Access Permissions

By default, all employees can create task templates. This permission can be restricted in the access settings.

{% note tip "User Documentation" %}

  - [How to set access permissions in Bitrix24 tasks](https://helpdesk.bitrix24.com/open/27846332/)

{% endnote %}
