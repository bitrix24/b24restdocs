> Scope: [`crm`](../../scopes/permissions.md)
>
> Who can execute the method: any user

```http
crm.status.fields()
```

The method returns the description of the fields in the directory.

## Parameters

No parameters.

## Example

```javascript
BX24.callMethod(
    "crm.status.fields",
    {},
    function(result)
    {
        if(result.error())
            console.error(result.error());
        else
            console.dir(result.data());
    }
);
```
