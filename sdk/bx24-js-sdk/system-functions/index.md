# Overview of System Functions

#|

|| [BX24.init](./bx24-init.md) | Adds an event handler for the "library is ready" event ||
|| [BX24.install](./bx24-install.md) | Allows setting an event handler for the "application is launched for the first time for the current user" event ||
|| [BX24.installFinish](./bx24-install-finish.md) | Function that signals the completion of the application's installer or setup process ||
|| [BX24.getAuth](./bx24-get-auth.md) | Retrieves the current authorization data via the OAuth 2.0 protocol ||
|| [BX24.refreshAuth](./bx24-refresh-auth.md) | Forces an update of the authorization key ||
